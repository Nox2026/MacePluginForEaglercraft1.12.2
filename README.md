HOW TO SET UP MACE PLUGIN FOR EAGLERCRAFT 1.12.2 (1.12.2 u2, 1.12.2 u3)

1. The first step is to have a TopEaglerServer (server) set up, if you already have one skip to step (_). To make a server go to https://topeaglerservers.com/ . Once on the website create an account. Nice, now you can make a server! At the top right of your screen you should see your account info, click on your username and look for a line that says MY SERVERS. Click it and you will be brought to a screen that should say no servers yet and click the create a server button. Go through the provided steps and once completed you should see the server bar. Now you have a server where you can click start and copy and paste the ip given to you in the add server on the multiplayer screen to play.
